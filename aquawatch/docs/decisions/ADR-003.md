# ADR-003: OpenStreetMap for Mapping

## Status
**Accepted** - Currently in use

## Date
2024-01-15 (Initial Development)

## Context
We needed an interactive map to display reported WASH issues across Bengaluru. The map should show:
- Location markers for each report
- Color-coded markers by category and status
- Clickable popups with report details

## Decision
Use Leaflet.js with OpenStreetMap tiles:
- **Library**: Leaflet.js 1.9.4
- **Tiles**: OpenStreetMap (free, no API key required)
- **Custom markers**: DIV icons with emoji and background colors

## Alternatives Considered

### Option 1: Google Maps API
- **Pros**: Better coverage, familiar API, offline maps
- **Cons**: Requires API key, billing setup, credit card
- **Rejected**: Too complex for prototype

### Option 2: Mapbox
- **Pros**: Beautiful tiles, good API
- **Cons**: Requires API key, has limits
- **Rejected**: Need to set up account

### Option 3: Static Map Image
- **Pros**: No API needed, simple
- **Cons**: Not interactive
- **Rejected**: Need interactive markers

### Option 4: CesiumJS (3D)
- **Pros**: 3D visualization
- **Cons**: Too heavy, overkill
- **Rejected**: Not needed for 2D markers

## Implementation Details

```javascript
// Initialize map
L.map('map', { zoomControl: false }).setView([12.9716, 77.5946], 12);

// Add tile layer
L.tileLayer('https://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png', {
    attribution: '© OpenStreetMap contributors',
    maxZoom: 19
}).addTo(mapRef.current);

// Custom marker with emoji
const icon = L.divIcon({
    html: `<div class="custom-marker" style="background: ${color}">${emoji}</div>`,
    iconSize: [50, 50],
    iconAnchor: [25, 25]
});
```

## Consequences

### Positive
- ✅ Free, no API key required
- ✅ No billing setup needed
- ✅ Good coverage of India/Bengaluru
- ✅ Easy to customize markers
- ✅ Lightweight library

### Negative
- ❌ Less detailed than Google Maps
- ❌ Occasional tile loading issues
- ❌ No street view
- ❌ No real-time traffic
- ❌ Dependence on OSM servers

## Custom Marker Design

Each marker shows:
- **Background Color**: Based on category (red for contaminated water, orange for broken facility, etc.)
- **Emoji Icon**: Category icon (💧, 🚽, 🕳️, 🧼)
- **Border**: White, 4px
- **Shadow**: Soft shadow for depth
- **Hover**: Scale up animation

## Future Considerations

For production, consider:
- **Mapbox Studio**: Custom styled tiles
- **Google Maps**: If needing India-specific features
- **Multiple tile layers**: Satellite, terrain
- **Cluster markers**: For dense report areas
- **Geocoding**: Convert address to coordinates

---

*Note: OpenStreetMap is free but has usage terms. For heavy production use, consider contributing to OSM or using a paid tile provider.*
